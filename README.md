## Laravel Admin
Laravel admin gives you admin template ready to integrate your system. It can be used as a starter template for your app. To get tarted, Follow the Steps or download this project and run.

## Installation
    - Install laravel project
    - Run composer require encore/laravel-admin 
    - Publish php artisan vendor:publish --provider="Encore\Admin\AdminServiceProvider"
    - Run php artisan admin:install
